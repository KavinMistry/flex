# Show: 
This is a simple and very lightweight show/flex plugin, which will also display the item itself upon hovering it's name in the chat! 

If you find any bugs please email me at: minekavincraft@gmail.com, I will fix them asap !

## Guide: 

### Commands:

```
/show 
```
Will display a message in chat, which a hoverable item display.
```
/flex
```
Alias for /show

### Permission:

```
show.command
```
default: true (Will allow all players to access this command unless a permission management plugin is installed)
